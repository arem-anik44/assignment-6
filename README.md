 Project Name  ---
 
DigiTools - Digital Product Store

Description  ----

This is a digital product website.  
User can see products and add to cart.  
Also can remove and checkout easily.  
Simple UI and clean design.

Technology Used  ----
- React JS  
- Vite  
- Tailwind CSS  
- DaisyUI  
- React Toastify  

Features  ----
- Add product to cart with success message  
- Remove item with warning toast  
- Checkout clear cart and show success
